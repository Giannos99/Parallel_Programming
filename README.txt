It is an implementation of Jon Conway's Game of Life, 
https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life
but with parallel programming
MPI, OpenMp, and Cuda programming were practiced.
Experiments were made for MPI and openMP parallel speed amelioration  using a 
network of 25 desktop personal computers.
Some sample results are in the ReadMeGreek pdf file.
